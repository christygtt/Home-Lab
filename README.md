# Home-Lab
A hands-on lab environment for experimenting with networking, cybersecurity, and systems administration. Includes virtual machines, scripts, and configurations for testing, learning, and building projects.

This repository hosts the configuration, automation scripts, and documentation for my personal homelab. The goal is to create a safe, isolated environment for:

Networking: Testing routers, firewalls, VLANs, and VPNs.

Cybersecurity: Practicing penetration testing, red team/blue team exercises, and security tools.

System Administration: Managing Linux and Windows servers, user management, and automation.

Virtualization: Running multiple virtual machines VirtualBox.

Automation & Scripting: Bash, Python, and PowerShell scripts for monitoring, backups, and lab management.

This homelab serves as both a personal learning environment and a reference for reproducible configurations. Contributions, ideas, and improvements are welcome!

Tech Stack / Tools:

Hypervisors: VirtualBox, VMware, Proxmox

OS: Ubuntu Server, Windows Server

Networking: pfSense, OpenVPN

Security Tools: Nmap, Metasploit, Wireshark, John the Ripper

Automation: Python, Bash, PowerShell
